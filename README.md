# Malicious URL Prediction

## Description
This project predicts whether a URL is malicious or safe using machine learning.

## Dataset
The dataset is from Kaggle:  
[Malicious URLs Dataset](https://www.kaggle.com/datasets/sid321axn/malicious-urls-dataset)

## How to Run
### Option 1: Manual Download
1. Go to the Kaggle dataset link above
2. Download the CSV file
3. Place it in the same folder as the `.py` file
4. Run your Python script:
```bash
python yourfile.py
